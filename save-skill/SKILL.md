---
name: save-result
description: saves the results of GRE-VOCABULARY-TEST. It creates two text files. First one includes list of the words which user answered them correctly. Second one               includes list of the words which user answered them incorrectly. 
---



## Instructions

1. **Creat 2 text files**:
   Create 2 text files. one named correct-answers.txt the other named incorrect-answers.txt in resources folder. If these files already exist in that folder delete       them and create new files. 
   
2. **Save the user answers**:
   After user submits answer for each question. If answer is correct add the word and its definition to correct-answers.txt. Otherwise add the word and its definition      to  incorrect-answers.txt
3. ** Save files**:
    save changes made to both files created in step 1.
